Libraries & Submodules Documentation

Project Directory Structure

  • /src/: Core application source files.
    • /components/: A-Frame specific entity-component logic.
    • /state/: Application state management via aframe-state-component.
    • /templates/: Nunjucks HTML templates.
  • /vendor/: Pre-compiled legacy dependencies not available via modern NPM.
  • /assets/: Sounds, textures, fonts, and 3D models.
  • /build/: Webpack output directory (contains build.js).

Submodules & External Libraries Tracking

Library / Submodule Version Location Description & Usage
A-Frame Core 0.8.2 (Modified) vendor/aframe-master.js The core WebVR framework used to render the game scene and handle the Entity-Component-System (ECS). Modified to preserve legacy API compatibility.
MediaPipe Hands ^0.4.x NPM (@mediapipe/hands) Used for optical hand tracking in Desktop mode. Maps 2D hand coordinates/depth to the 3D saber positions.
TMI.js ^1.8.3 NPM (tmi.js) Core library for Twitch Chat integration. Listens to chat commands (e.g., !vote, !speedup) and emits events to the game state.
Firebase ^5.5.8 NPM (firebase) Required for leaderboard cloud store operations and high score persistence.
Nunjucks ^3.2.x NPM (nunjucks) HTML templating engine used via Webpack to dynamically build the UI (e.g., src/templates/menu.html).
