# Ideas

## Refactoring / Code Health
- Migrate from Nunjucks/Webpack to Vite for vastly superior development speed.
- Update A-Frame to latest versions (WebVR -> WebXR transition).
- Convert core game logic to TypeScript for better type safety.

## Gameplay Enhancements
- **Fitness Tracker Integration**: Read heart rate via Bluetooth Web API.
- **Dynamic Lighting**: Make stage lights react to Twitch chat activity.
- **Custom Saber Importer**: Allow dragging and dropping `.obj` or `.gltf` files directly into the browser to change saber models.

## Twitch Integration Concepts
- **Chat vs Player**: Viewers spend channel points to trigger bad modifiers (e.g., fast song, ghost notes) for 10 seconds.
- **Hype Train Mode**: If chat spams enough emotes, activate "Star Power" multiplier.
